# SpiritForms 1.0

SpiritForms 1.0 is a nuBuilder-based application for managing paranormal investigation cases, investigations, client access, uploaded evidence references, and completed reports.

This public repository contains the SpiritForms installation package, documentation, required database setup files, Chrome extension files, and release notes.

## Important

Do not commit live client data, production database backups, server credentials, email credentials, or live `nuconfig.php` files.

The required clean installation `.sql` file is intentionally included because it is required to set up the SpiritForms database during installation.
